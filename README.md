# BPS Hotel Bookings Analysis Project

##  Project Overview
This project is developed for the **Business Project Support (BPS)** unit.  
The main purpose of this project is to analyze hotel booking data to understand customer behavior, booking patterns, and business insights using data analysis and visualization tools.

The project follows a complete data analysis process including data collection, cleaning, exploratory data analysis (EDA), and dashboard preparation.

---

##  Project Objectives
- To analyze hotel booking data from a business perspective  
- To clean and prepare raw data for analysis  
- To identify trends such as booking cancellations, customer types, and seasonal patterns  
- To prepare data for visualization using Power BI  
- To support business decision-making using data insights  

---

##  Dataset Description
The project uses real hotel booking data with the following files:

- **hotel_bookings_uncleaned.csv**  
  Raw dataset before cleaning  

- **hotel_bookings_cleaned.csv**  
  Cleaned dataset after handling missing values and errors  

- **hotel_bookings_for_powerbi.csv**  
  Final dataset prepared specifically for Power BI dashboard  

---

##  Analysis Files
- **hotelbookingEDA.ipynb**  
  Performs Exploratory Data Analysis (EDA) such as:
  - Data overview
  - Missing value checking
  - Statistical summaries
  - Business-related insights  

- **hotelbooking.ipynb**  
  Main analysis notebook including:
  - Data loading
  - Data cleaning
  - Feature processing
  - Final outputs for reporting
